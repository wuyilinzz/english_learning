# 1 动词分类

## 1.1 可作谓语动词
- 实义动词：有实实在在动作意思的词（跑、吃、看、做）
	- 按动作承受者划分：及物动词、不及物动词；
	- 按状态动作划分：动作动词（eat）、状态动词（have、own）；
	- 使役动词：表示让、使、叫、令、请、迫使，让别人做某事的动词。
		- let / make / have sb do（不带to）
		- get sb to do
	
- 联系动词：连接主语和表语，不表具体动作

## 1.2 可辅助构成谓语动词

### 1.2.1 助动词
定义：帮助主要动词构成各种时态、语态、否定句和疑问句的词。它们本身没有实际意义（或者说意义很虚）。
常见助动词：be/do/have。

### 1.2.2 情态动词
定义：表示说话人的情感或态度（如可能性、必要性、允许、能力、意愿）。它们后面必须加动词原形。
常用情态动词：can/could， may/might， must， shall/should， will/would， ought to， need

## 1.3 不可作谓语动词
- 非谓语动词
不定时、过去分词、现在分词、动名词；

# 2 动词时态
- 定义：时间 + 状态；
- 时间分类：现在、过去、将来、过去将来；
- 状态分类：一般、完成、进行、完成进行；
排列组合共16种。

## 2.1 动词时间
按时间线划分大概是下面这样子，前3个挺好理解，就过去将来有点绕，简单讲就是过去的时间点讨论将来，但这个将来的时间点，又比现在还早。
比如下面，我在前2天讨论前一天要干什么，但前一天还是比今天早，所以是过去将来？

---过去---过去将来---现在---将来--->

## 2.2 动词状态
进行：兔子“正在吃”萝卜；
完成：兔子把萝卜“吃完了”；（完成状态并不能表示动作什么时间完成了）
完成进行：兔子从”早上开始吃”胡萝卜，现在“还没停”；（简单理解为做一半，还没做完？）
一般：兔子一般吃胡萝卜；（没有具体的状态信息）

## 2.3 动词时间 + 状态组合
一共有16种，比较好的区分方法就是，把4个状态，在时间线上来回移动。

## 2.4 动词难点
- 时 & 态分不清

- 动词变位
	动词时态还要根据情况做变化，比如eat表过去还得变成ate等等。

- 助动词
	也需要变位。
	
## 2.5 时态16种组合分类

|          | 一般   | 进行   | 完成   | 完成进行 |
|:-------- |:------ |:------ |:------ |:-------- |
| 现在     | am/is/are/do/does | am/is/are doing | have/has done | have/has been doing |
| 过去     | was/were/did | was/were doing | had done | had been doing |
| 将来     | be going to do;<br>will/shall do; | will/shall be doing | will/shall have/has done | will/shall have been doing |
| 过去将来 | was/were going to do;<br>would/should do; | would/should be doing | would/should have done | would/should have been doing |

# 3 动词语气
## 3.1 陈述语气
定义：描述现实。
The rabbit ate the carrot.

## 3.2 祈使语气
定义：命令/请求。
用法：
	第一步，用第二人称（“你”）用一般现在时态。
	第二步，去掉“你”，并保证动词原型。eat the carrot!
例子：
	you eat the carrot.--->eat the carrot!
	you don't eat it.--->Don't eat it!
	You are quiet!--->Be quiet!

## 3.3 虚拟语气
定义：说的不是真事，是假设、愿望、建议、后悔、不可能发生的事。

1. 与现在事实相反（现在不可能）
2. 与过去事实相反（过去已经发生，后悔 / 假设）
3. 与将来事实相反（将来不太可能）

### 3.3.1 与现在事实相反（现在不可能）
结构：If + 主语 + 过去式（be 统一用 were），主语 + would/could + 动词原形

上面那个是从句 + 主句。

例句：
- If I were you, I would go.
	如果我是你，我就去。（我不可能是你）
- If I had time, I would help you.
	我要有时间就帮你。（实际没时间）

### 3.3.2 与 过去 事实相反（最常考）
结构：If + 主语 + had done，主语 + would/could have done

例句：
- If you had come earlier, you would have met him.
	你早点来就能见到他了。（实际来晚了）
- I would have passed if I had studied harder.
	当初努力点我就过了。（实际没努力）

### 3.3.3 与 将来 事实相反（可能性极低）
结构：If + 主语 + should do /were to do，主语 + would do

例句：
- If it should rain tomorrow, we would cancel the trip.
	万一下雨，我们就取消。（不太可能下雨）
- If I were to win the lottery, I would buy a house.
	我要是中彩票就买房。（几乎不可能）

### 3.3.4 省略 if 的倒装（高级写法）
把 were / had / should 提前，去掉 if：

- Were I you, I would go.
= If I were you…
- Had you come earlier, you would have met him.
= If you had come earlier…
- Should it rain tomorrow, we would cancel.

### 3.3.5 表示 “建议、命令、要求” 的虚拟
一个规则：suggest / advise / propose / demand / require / request / insist / order后面的 that 从句：(should) + 动词原形

例句：
- I suggest that he (should) go there.
- The doctor advised that she (should) rest.
- He insisted that I (should) be present.

注意：suggest 表 “暗示” 时不用虚拟；insist 表 “坚持说（事实）” 时不用虚拟。

### 3.3.6 wish /if only（愿望、后悔）
- 对现在：wish + 过去式
I wish I were rich.
- 对过去：wish + had done
I wish I had studied harder.
- 对将来：wish + would/could do
I wish it would stop raining.

- if only 和 wish 一样，只是语气更强：
If only I had known earlier!
要是我早点知道就好了！

### 3.3.7 would rather /as if /as though
- would rather + 过去式（现在 / 将来）
I would rather you came tomorrow.

- would rather + had done（过去）
I would rather you hadn’t done that.

- as if /as though + 过去式（现在不真实）
He talks as if he knew everything.

- as if + had done（过去不真实）
He talks as if he had seen it.

### 3.3.8 总结

| 假设类型       | If 从句         | 主句                  | 例句                                  |
|----------------|-----------------|-----------------------|---------------------------------------|
| 与现在相反     | 过去式 / were   | would do              | If I were you, I would go.            |
| 与过去相反     | had done        | would have done       | If I had known, I would have helped.  |
| 与将来相反     | should do       | would do              | If it should rain, we would stay.     |
| 建议命令要求   | (should) do     | —                     | I suggest that he (should) go.        |
| wish 现在      | 过去式          | —                     | I wish I were young.                  |
| wish 过去      | had done        | —                     | I wish I had come.                    |

- 现在假 → 用过去
- 过去假 → 用过去的过去（had done）
- 将来假 → 用 should /were to

# 4 非谓语动词
- 定义：不受主语限制，不具备表达时间和人称的本领。

## 4.1 动词不定时
**规则：** to + 动词原型

- 1.作主语
	- **To eat a carrot** everyday is good for the rabbit.（It is good that the rabbit eats a carrot everyday.  it's good for sb to do sth固定搭配）

- 2.作宾语
	- The rabbit likes **to eat** carrots.（注意有些动词后面是不能接不定式的）

- 3.作宾语补语
	- The rabbit expected the wolf **to eat a carrot**.
	- Carrots make the rabbit (省略to) feel happy.（see/find/watch/have/make/let这类使役动词后面用不定式作宾补要**省略to**）

- 4.作表语
	- The rabbit's dream is **to eat** every kind of carrot in the world.

- 5.作定语
	- The rabbit has a lot of carrots **to eat**.
	
- 6.作状语
	- I was suprised **to get** a thumbs-up.（谓语动词后面+to do）

- 7.高级不定时：to + 动词原型（"状态"）
	- to be eating a carrot/to have eaten a carrot/to have been eating a carrot.
	- The rabbit seemed to be eating a carrot.
	
## 4.2 动名词
**规则：** 动词变ing形式，然后当名词/表语等用

- 1.作主语
	- Eating carrots is healthy for the rabbit.

- 2.作表语
	- The rabbit's hobby is growing carrots.

- 3.作定语
	- a sleeping pill. = a pill for sleeping.(具有什么样的性质/功能，一片药用来睡觉(功能)，和一片正在睡觉的(形容词)药，意思完全不同）
	
- 4.作宾语
	- The rabbits likes eating carrots.
	- The rabbit enjoys eating carrots.（这里enjoy后面不能跟to do，类似的还有avoid/advise/consider/escape/mind/practice/insist on/give up/feel like/put off等）

- 5.介词 + 动名词
	- The rabbit is fond of eating carrots.
	- The rabbit is looking forward to seeing the wolf again.（这里别把to当成不定时的to去理解，这里的to是介词，look forward to doing sth）
	
- 6.forgot to do和forgot doing区别
	- forgot to do = 忘记做  （没做）
	- forgot doing = 忘记做过	（做了）
	
## 4.3 现在分词
**规则：** 动词变ing形式，然后当形容词用

- 1.作表语
	- The rabbit is charming.

- 2.作宾补
	- The wolf saw the rabbit eating a carrot.（现在分词作宾补，表示"正在"）
	- The wolf saw the rabbit eat a carrot. （不定时(省略to)作宾补，表示"全程"）

- 3.作状语
	- 表示时间、原因、条件、结果、让步、目的、方式和伴随情况等
	- Hearing the news,the rabbit became excited.

## 4.4 过去分词
**规则：** 动词变过去式，然后当形容词用，和现在分词当形容词的区别在于，过去分词表示被动的，例如minced（被搅碎） carrot

- 知识点补充
	- 定语：修饰名词的词，用来限定、形容这个名词
	- 前置定语：修饰词放在名词前面，a beautiful girl
	- 后置定语：修饰的一坨词（短语：过去分词 + 状语/补语）放在名词后面，The bike made in China is cheap.

- 1.作定语（后置定语）
	- 过去分词短语(eaten by)作定语，要放到被修饰的名词(The carrot cake)之后
	- The carrot cake eaten by the rabbit is delicious.

- 2.作表语
	- The rabbit is interested in growing carrots.(sb.is interested in doing sth)
	- The rabbit was frightened.兔子被吓到了。
	- The rabbit was frightening.兔子令人感到害怕。
	
- 3.作宾补
	- The rabbit found his carrot stolen.
	
- 4.作状语
	- Seen from the hill,the carrot field looks beautiful.
	- 如何区分是用现在分词还是过去分词？ 答：看是想表达主动(现在分词)还是被动(过去分词)，例如上面的山丘，它是被人看的而不是主动看别人，所以这里用过去分词。

# 5 独立主格

## 5.1 定义

**概念：** 自带逻辑主语的短语，不是完整句子，不作从句，只作状语（表时间、原因、条件、伴随、方式）。

**公式：** 名词/代词 + 非谓语/形容词/副词/介词短语

**特点：**
	- 有自己独立的主语，和主句主语不一样
	- 没有连词、没有谓语动词
	- 只修饰整个主句，放句首 / 句中 / 句尾都可以
	
## 5.2 结构 

- 1.名词 + 现在分词 doing
	- The weather being fine, we went out.

- 2.名词 + 过去分词 done
	- The work finished, we went home.

- 3.名词 + 不定式 to do
	- No one to talk with, he felt lonely.

- 4.名词 + 形容词
	- He came in, face pale.
	
- 5.名词 + 副词
	- The meeting over, everyone left.

- 6.名词 + 介词短语
	- The teacher entered, book in hand.
	
## 5.3 和从句的区别

- 状语从句：有连词 + 完整主谓
	- When the weather was fine, we went out.
	
- 独立主格：无连词、无谓语，只有「名词 + 短语」
	- The weather being fine, we went out.

## 5.4 注意事项

- 主句和独立主格主语不同
- 不能加 because/when/if 这类连词
- 逗号隔开，不作句子，只是短语

# 6 助动词&情态动词

**作用：** 本质同一类，帮动词凑完整意思、凑时态、凑语气。本身不能单独做实义谓语，都要依附后面的实义动词

**区分：**
 
	1. 助动词本身无实义，帮忙构成时态、语态、否定、疑问。注意，有些助动词本身可以作实义动词（单独作谓语），例如be/do/have，I have a pen.
	2. 情态动词，本身有词义，表语气、态度、推测、能力、许可
	
## 6.1 助动词

- 基本助动词：be/do/have
- 半助动词：be able to/be going to/be about to/had better + do

- 例句：
	- be (am/is/are/was/were) = He is reading（进行时）/It was made（被动语态）
	- do/does/did = Do you like it?/I don't know/I did tell you 帮提问、否定、强调
	- have/has/had = I have finished.
	
## 6.2 情态动词

- 常见：can/could、may/might、will/would、shall/should、must、ought to、dare/dared、need、used to

- 规则：
	- 后面直接加动词原形，不加 to（ought to 除外）
	- 第三人称不加 s
	- 没有不定式、分词、过去式变形（could/would 是委婉式，不是过去式）
	
- 例句：
	- You should study.
	- I would like to introduce the new product to you.
	
# 7 实义动词&系动词

## 7.1 实义动词

- 实义动词：有实实在在动作意思的词（跑、吃、看、做）
	- 按动作承受者划分：及物动词、不及物动词；
	- 按状态动作划分：动作动词（eat）、状态动词（have、own）；
	- 使役动词：表示让、使、叫、令、请、迫使，让别人做某事的动词。
		- let / make / have sb do（不带to）
		- get sb to do

- 例如：
	- I'll have wolf wash the dishes.这里的have不是助动词而是使役动词的用法，我让狼去刷碗

## 7.2 系动词

**作用：** 联系主语和表语，后面不接宾语，接表语。本身没多少实际动作，只起 “等于 / 状态” 作用

- be动词： am/is/are/was/were
- 感官系动词：look, sound, smell, taste, feel
- 变化系动词：become, get, turn, grow

## 7.3 怎么区分 have 是「使役动词 / 助动词 / 实义动词」

- 后面是 人 + 动词原形 : 使役动词（让某人做）
- 后面是 某物 + 过去分词 ： 使役被动 （被别人怎么样）
- 后面是 过去分词 done : 助动词（完成时态）
- 后面直接接 名词 : 实义动词（有 / 吃 / 喝）

- 例句：
	- 使役：I have him go home. 让他回家
	- 助动：I have gone home. 我已经回家了
	- 实义：I have a book. 我有一本书
	- 使役被动：I have my hair cut.我把头发剪了（让别人给自己剪）
