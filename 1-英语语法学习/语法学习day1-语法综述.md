# 1 简单句

概念：把一个句子拆分到不能再拆的基本句子，称为简单句。

## 1.1 句子构成

所有的英语句子，最后总结起来就是两个组成部分（句子成分clause elements）：什么 + 怎么样。

  人/物			 verbs动作/发生了什么
主语（subject） + 谓语（predicate）
  什么			   怎么样。

### 1.1.1 句子成分

1.主语
2.谓语动词
3.宾语
4.宾语补语
5.主语补语（表语）
6.定语（修饰主语或宾语）
	the little white rabbit ate a large carrot.
	the little white是主语rabbit的定语。
	a large是宾语carrot的定语。
7.状语（修饰谓语动词）
	the rabbit ate quickly.
	quickly是ate的状语。
8.同位语（把主语、宾语重复描述）
	Anon,an engineer,eats meats.

## 1.2 动词

### 1.2.1 动词-句型

1.可以独立完成的动作
句型：主语 + 不及物动词
例子: Anon sleeps. 爱音睡觉。
解释：这种没有承受者的动词叫不及物动词，intransitive verbs	。

2.有1个动作的承受者
句型：主语 + 单及物动词 + 宾语（动作的承受者）
例子：Anon hates you. 爱音讨厌你。
解释：这种需要一个宾语跟着动词后面，不然意思不完整，例如爱音讨厌，讨厌啥或什么，得有一个宾语。transitive verbs。

3.有2个动作承受者
句型：主语 + 双及物动词 + 间接宾语 + 直接宾语
例子：Anon teaches you English. 爱音教你英语。
解释：这种语句只有一个宾语不完整，爱音教你，教你啥？得补充清楚。

4.只有一个动作承受者（但需补充）
句型：主语 + 复杂及物动词 + 宾语 + 补语。
例子：Anon considers you stupid. 爱音认为你笨。
解释：这里有一个宾语但是不完整，爱音认为你，认为你啥？得补充清楚，但是又不像句型3是另外一个承受者。complex transitive verbs。
这样的动词，必须要有个补充承受者的信息才意义完整。补充的信息叫`补足语-补语-宾补-complement-object complement`。

5.非"动作"
句型：主语 + 联系动词 + 主语补语（表语）
例子：Anon looks tall.	爱音看起来高。
解释：is这类把后面的词与前面的名词联系起来，这类此叫`联系动词-linking verbs`。

6.there be句型
there is a rabbit。可理解为句型5的倒装。

7.主语 + 谓语动词 + 状语
i live in China。可理解为句型1的延伸。

8.主语 + 谓语动词 + 宾语 + 状语
i put the carrot on the table。可理解为句型4的延伸。

注：谓语 ≠ 谓语动词。主语后面剩余的内容就是谓语，谓语动词只是谓语里的一部分。

## 1.2.2 谓语动词/非谓语动词

1.核心区别

| 谓语动词 | 非谓语动词 |
|----------|------------|
| 本质：句子的“心脏” | 本质：句子的“零件” |
| 功能：表示动作/状态，构成句子主干 | 功能：充当其他成分（名词、形容词、副词） |
| 标志：有时态、人称、数的变化 | 标志：不随主语变化，形式固定 |

2.详细对比表格

| 对比维度 | 谓语动词 | 非谓语动词 |
|----------|----------|------------|
| 定义 | 在句子中充当谓语的动词，说明主语“做什么”或“是什么” | 不能单独作谓语的动词形式，充当其他句子成分 |
| 形式 | 随时态（do/did/will do）、人称（I am/He is）、数（He runs/They run）变化 | 三种固定形式：<br>• 不定式 (to do)<br>• 动名词 (doing)<br>• 分词 (doing/done) |
| 语法特征 | • 有时态和语态<br>• 有语气（陈述/虚拟）<br>• 与主语保持一致 | • 无时态变化<br>• 不受主语影响<br>• 但有语态（主动/被动）和时态关系（一般/完成） |
| 句子中的位置 | 紧跟主语之后，构成句子主干 | 可出现在主语、宾语、定语、状语、补语位置 |
| 能否单独成句 | 能。一个完整的句子必须有谓语动词 | 不能。非谓语动词短语只是句子的一部分 |
| 标志词 | 常与情态动词（can/must）、助动词（have/be）连用 | 常与介词、冠词连用，或单独出现 |

3. 具体例子对比
- 同一个动词的不同身份（以动词 "swim" 为例）

| 句子 | 动词形式 | 类型 | 说明 |
|------|----------|------|------|
| I swim every day. | swim (原形) | 谓语动词 | 主语I，时态一般现在时 |
| Swimming is good for health. | Swimming (动名词) | 非谓语 | 作主语 |
| I want to swim this afternoon. | to swim (不定式) | 非谓语 | 作宾语 |
| The swimming man is my brother. | swimming (现在分词) | 非谓语 | 作定语修饰man |
| I saw him swim across the river. | swim (不带to的不定式) | 非谓语 | 作宾补 |

- 句子对比

| 句子 | 谓语动词 | 非谓语动词 | 解析 |
|------|----------|------------|------|
| He reads a book. | reads | 无 | 只有一个动词，作谓语 |
| He wants to read a book. | wants | to read | wants是谓语，to read是宾语 |
| Reading a book, he fell asleep. | fell | Reading | fell是谓语，Reading是状语 |
| The book read by him is interesting. | is | read | is是谓语，read是后置定语 |

注意：一个独立分句/简单句 只能有一个谓语动词，但一个复合句/复杂句可以有多个谓语动词（每个分句一个）。

## 1.2.3 谓语动词的三大本领
1.动作时间
现在、过去、将来、过去将来；
2.动作状态
一般、完成、进行、完成进行；
上面的按照排列组合就有16种。
3.动作假设，情感
虚拟语气：if i were a rabbit。
陈述语气：i ate a carrot and ... 
祈使预期：Eat this carrot and ... 

## 1.2.4 助动词
英语里的谓语动词往往不能独立完成"三大本领"，另外谓语动词独自也无法完成比如表示否定，可能性，必须性等意思，这时候就需要助动词来帮忙完成任务。

以eat举例：
（协助）表示"吃过了（完成）" 	-> have 	-> have eaten
（协助）表示"正在吃（进行）"  	-> be   	-> is eating
（协助）表示"被吃" 		 	-> be   	-> is eaten
（协助）表示"有能力吃" 		 	-> can   	-> can eat
（协助）表示"有可能吃" 		 	-> might   	-> might eat
（协助）表示"有义务吃" 		 	-> must   	-> must eat
（协助）表示"不吃（否定）" 		-> do   	-> do not eat

# 3 复杂句
句子里面套另外一个句子，complex sentece。

句子 = 主句（Main clause） + 从句（subordinate clause）

从句：把简单句修改一下充当另一个句子的句子成分。

把一个简单句做下面几种语句，能分出几种从句：
1.名词性从句：
作主语----->主语从句 
作宾语----->宾语从句
作表语----->表语从句
作同位语--->同位语从句
2.定语从句
作定语--->定语从句
作状语--->状语从句

# 5 总结概括
	 
- 动词：谓语动词（不及物、单及物、双及物、复杂及物、系动词）、助动词、非谓语动词

- 句子成分：
主语 + 动词
主语 + 动词 + 宾语
主语 + 动词 + 间接宾语 + 直接宾语
主语 + 动词 + 宾语 + 宾补
主语 + 动词 + 补语/表语
其他的还有定语、状语、同位语。

- 简单句派生复合句（并列复合句）、复杂句（主从复合句）。复杂句里面分主句和从句。

- 词类：名词、冠词、代词、形容词、数词、副词、介词、叹词、连词、动词。
